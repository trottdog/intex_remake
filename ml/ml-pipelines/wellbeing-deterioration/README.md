# Wellbeing Deterioration

This folder contains the standardized predictive and explanatory notebook templates for `wellbeing_deterioration`.

## Notebook Standard

- Decision support: Document the wellbeing-deterioration concept while the current future-health target remains too sparse for a strong production model.
- Target summary: Notebook-only track: `label_wellbeing_deterioration_next_90d` exists in the shared table, but the eligible future-health sample is still sparse enough that this phase keeps the work explanatory-first.
- Recommended widgets: `risk_badge_widget`, `insight_summary_card`, `recommendation_panel`
- Primary users: case managers, health and wellbeing staff
