# Home Visitation Outcome

This folder contains the standardized predictive and explanatory notebook templates for `home_visitation_outcome`.

## Notebook Standard

- Decision support: Support reintegration planning with an early signal about which home environments look more supportive.
- Target summary: Current predictive label: `label_supportive_home_visit_next_120d`, using future visitation outcomes with favorable, low-concern, cooperative-home signals.
- Recommended widgets: `insight_summary_card`, `recommendation_panel`, `ranked_table_widget`
- Primary users: case managers, reintegration staff
