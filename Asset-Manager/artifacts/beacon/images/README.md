# Images folder

This directory contains image assets used by the app.

## Conventions

- Keep **currently referenced** public assets at the root so existing URLs like `/images/<file>` keep working.
- Put other categorized assets in subfolders (e.g., `directors/`, `headshots/`, `location/`, `support/`).
- Move unused/experimental assets into `archive/` (dated subfolders) so the root stays clean.

## Archive

- `archive/unused-2026-04-09/` contains files that were not referenced anywhere in the repo at the time they were moved.
